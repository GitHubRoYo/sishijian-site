# Assets Sources & Licenses

## Brand / Partner Logos (42 SVGs)

All 42 brand logos are located in `/public/assets/logos/brands/`. They are **custom-created SVG files** using white (#ffffff) fills on transparent backgrounds, designed for display on dark section backgrounds.

### Design Approach

- **Social media icons** (Facebook, Instagram, WhatsApp, Google, TripAdvisor, LINE, Yahoo): Created using SVG paths that represent the recognizable brand marks. These are simplified iconic representations.
- **Chinese media brands** (明報, 星島日報, 東方日報, etc.): Clean Chinese character wordmarks using serif font stacks, reflecting the traditional newspaper logotype style.
- **English media/platform brands** (am730, hket, on.cc, etc.): Clean sans-serif or serif wordmarks matching brand character.
